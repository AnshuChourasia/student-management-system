# 🎓 Student Management System

A menu-driven Student Management System built in Python to practice core programming concepts such as dictionaries, functions, loops, exception handling, and CRUD operations.

## 📌 Features

- ➕ Add a new student
- 📋 View all students
- 🔍 Search a student by ID
- ✏️ Update student details
- ❌ Delete a student
- 📊 Calculate average marks
- 🏆 Find the highest scorer
- 👥 Count total students
- 📝 Grade evaluation

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Practiced

- Functions
- Dictionaries (Nested Dictionaries)
- Loops
- Conditional Statements
- Exception Handling (`try` / `except`)
- CRUD Operations (Create, Read, Update, Delete)
- Menu-Driven Programming
